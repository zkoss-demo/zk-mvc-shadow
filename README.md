# zk-mvc-shadow

Example source code for the mvc-shadow smalltalks

## Build commands

build war (output target/mvc-shadow.war)
```
mvn clean package
```

run with jetty
```
mvn jetty:run
```
http://localhost:8080/mvc-shadow
